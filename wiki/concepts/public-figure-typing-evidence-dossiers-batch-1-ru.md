# Стартовые evidence dossiers для глубокого типирования — Санников, Романюк, Бандура

Статус: **нулевой слой глубокого dossier**, не финальное уточнение типов. Документ переводит первые три приоритетные фигуры из общего списка в формат [[public-figure-typing-evidence-protocol-ru]]. Здесь уже есть source set, первые evidence rows и differentiator plan, но ещё не хватает полного корпуса проповедей/интервью/Q&A для повышения уверенности.

## Почему выбраны эти три фигуры

1. **Сергей Санников** — больше всего письменных и образовательных материалов; лучший кандидат для проверки структуры мышления.
2. **Николай Романюк** — есть церковный контекст ИБЦ и вероятный корпус проповедей; важная спорная пара `ЭИИ vs ЛИИ`.
3. **Игорь Бандура** — есть сильный тематический материал о духовном отцовстве; важно проверить, не переобучились ли мы на одну тему.

| Игорь Бандура | ЭИЭ / ЭВЛФ / БПВН | ЭСЭ / ВЭЛФ / ПБВН | духовное отцовство как личный паттерн vs тематический жанр |

- `ЛИИ vs ИЛИ`: в лекциях он чаще строит классификацию/понятийную схему или разворачивает историческую динамику/вероятностную траекторию?

- `ВПБН vs ВБПН`: история традиции или будущее богословского образования является главным творческим способом раскрытия вечностного смысла?

- один материал о конфликте/кризисе/решении в образовательной или церковной сфере.

## Общие следующие шаги

1. Найти и сохранить ссылки на 3–5 полных материалов по каждому человеку.
2. Для каждого материала создать 5–10 evidence rows.
3. Сформировать blind packets без имени/должности.
4. Прогнать независимое кодирование.
5. Пересчитать confidence model.
6. Обновить main catalogue только после того, как есть реальные цитаты/фрагменты, а не только source set.

## Минимальный критерий для следующего повышения уверенности

- Сергей Санников: минимум 2 лекции/интервью с живой аргументацией + 1 письменный фрагмент.
- Николай Романюк: минимум 3 проповеди ИБЦ + 1 интервью/Q&A.
- Игорь Бандура: минимум 3 материала вне темы духовного отцовства.
