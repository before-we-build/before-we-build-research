# Методологическое v2-перетипирование популярных баптистских фигур Украины

Статус: **второй проход после методологического аудита**, не окончательное типирование. Этот документ применяет новые требования из [[public-figure-typing-methodology-audit-ru]] к текущему списку из [[baptist-audience-public-figure-typing-ru]]:

- сначала оценивать метод и силу источников;
- кодировать шкалы / оси / признаки до четырёхбуквенного типа;
- отдельно учитывать role-overfit;
- понижать уверенность, если есть только официальная биография;
- по темпористике требовать не просто темы прошлого/будущего/вечности, а позиционный режим;
- LLM/text-analysis использовать только как извлечение признаков, не как финальное решение.

## Method scout

| Источник | Сила для фактов | Сила для типа | Главный риск |
|---|---:|---:|---|
| Официальная страница руководства ВСЦ ЕХБ | высокая | низкая | должность и жанр биографии имитируют тип |
| ВРЦиРО по Валерию Антонюку | высокая для роли | низкая | представительский профиль, не живой стиль |
| Материал о духовном отцовстве Игоря Бандуры | средняя | средняя | одна тема может переобучить на ЭИЭ/1Э/1Будущее |
| Персональный сайт Сергея Санникова | высокая для корпуса трудов | средняя | показывает продукты и сферы, но не весь стиль взаимодействия |
| Сайт Ирпенской библейской церкви | высокая для роли Романюка | низкая-средняя | церковный сайт показывает служебную экосистему |
| Wikipedia по Мохненко | средняя | низкая-средняя | энциклопедическая сводка и медийный образ |

Методический gate:

- только официальная биография → максимум `низкая`;
- биография + один тематический материал → максимум `низкая-средняя`;
- корпус книг/статей/сайта + биография → можно `низкая-средняя`, иногда `средняя по направлению`, но не по точному коду;
- темпористика почти всегда ниже по уверенности, если нет материалов о личном переживании времени, выбора, смысла и кризиса.

## Арбитраж по фигурам

### Валерий Антонюк

**Method scout.** Доступны официальная биография ВСЦ ЕХБ и представительская страница ВРЦиРО. Это сильные источники для факта лидерства, но слабые для личного стиля. По новому gate максимум уверенности — `низкая`.

**Scale coding before type.**

- Соционика: заметны `Te/Si`-подобные признаки организации, служебной устойчивости и координации; но это может быть функция председателя. Нормативно-структурный `Ti/Se`-след тоже возможен, поэтому `ЛСИ` поднят как более важная альтернатива, чем раньше.
- Психософия: Воля/Логика высоко выглядят из роли; порядок `ВЛ` vs `ЛВ` неразличим без живых материалов.
- Темпористика: Прошлое/Будущее видны как преемственность братства и планирование Союза; но это институциональная рамка, не личный temporal mode.

**V2-итог.** `ЛСЭ / ВЛФЭ / ПБНВ`, но только низкая уверенность; альтернативы `ЛСИ`, `ЭСИ`, `ЛВФЭ`, `БПНВ` обязательны.

### Игорь Бандура

**Method scout.** Есть официальная биография и один сильный тематический материал о духовном отцовстве. Это лучше, чем только биография, но один тематический текст может переобучить гипотезу.

**Scale coding before type.**

- Соционика: устойчивые признаки `Fe/Ni`: поколение, вдохновение, передача духа, личный пример, будущее служения. Альтернатива `ЭСЭ` остаётся, если в других материалах больше общинной атмосферы и практической заботы.
- Психософия: высокие Эмоция/Воля выглядят вероятно; `ЭВ` объясняет эмоционально-смысловую подачу и процессное выращивание лидеров; `ВЭ` остаётся, если первична рамка ответственности.
- Темпористика: Future/Past читается сильнее всего: будущее поколения через передачу прошлого опыта. Но нужно проверить материалы вне темы отцовства.

**V2-итог.** `ЭИЭ / ЭВЛФ / БПВН`, уверенность `низкая-средняя`, не выше.

### Сергей Санников

**Method scout.** Источников больше: энциклопедические биографии, персональный сайт, корпус книг/статей, образовательные и аккредитационные роли. Это самый сильный набор в текущем списке, но он всё равно лучше показывает продукты мысли, чем живую динамику выбора и конфликта.

**Scale coding before type.**

- Соционика: сильные `Ti/Ne` или `Ti/Ni` признаки: систематизация богословия, истории, сакраментологии, редакторская работа, учебные материалы. `ИЛИ` остаётся, если в текстах больше исторической динамики и вероятностных траекторий, чем структурной классификации.
- Психософия: `1Л` наиболее устойчиво; `2В` объясняет институциональное строительство. `ЛЭВФ` остаётся, если педагогический контакт и просветительская эмоциональность окажутся сильнее волевого процессного слоя.
- Темпористика: `ВП` поддерживается темами богословского смысла и истории. Но у богослова Вечность жанрово загрязнена, поэтому точный порядок В/П/Б требует живого сравнения материалов.

**V2-итог.** `ЛИИ / ЛВЭФ / ВПБН`, уверенность `низкая-средняя`; можно говорить о средней уверенности только по направлению, не по точному коду.

### Геннадий Мохненко

**Method scout.** Не баптист в узком смысле; включён как более широкая украинская евангельская фигура. Открытая база в текущей странице опирается в основном на энциклопедический и медийный образ.

**Scale coding before type.**

- Соционика: сильные action/contact traces: публичная мобилизация, защита детей, социальное служение, капелланство. Это поддерживает `СЭЭ`, но `СЛЭ` остаётся, если первична силовая организация, а `ЭИЭ` — если первична миссионерская драматургия.
- Психософия: `1В` вероятна по масштабу инициатив, но это может быть роль активиста. `2Ф` vs `2Э` неразличимо без длинных интервью: практическое действие и эмоциональная мобилизация оба сильны.
- Темпористика: `1Будущее` вероятно по миссии и проектам вроде «мир без сирот», но медийная формула может усиливать future-frame.

**V2-итог.** Сохранить `СЭЭ / ВФЭЛ / БНПВ`, но понизить до `низкая` в основной таблице, пока нет более полного source dossier.

### Владимир Кондор

**Method scout.** В текущем проходе есть почти только официальная биография. По новым gates максимум `низкая`.

**Scale coding before type.**

- Соционика: `ЛСЭ` поддерживается исполнительной, академико-административной и миссионерской координацией; `ЛИИ` остаётся из-за академического деканата и богословского образования.
- Психософия: `ЛВ` vs `ВЛ` почти равны: образование и деканат дают Логику, исполнительная роль — Волю.
- Темпористика: `ПБ` выглядит как преемственность + будущие задачи, но это биографический маршрут, не temporal heatmap.

**V2-итог.** Без смены primary, но только `низкая`.

### Роман Вечерковский

**Method scout.** Официальная биография описывает миссию, новые церкви, мужские группы, возвращение в Мариуполь после 2014 года. Сильный кризисный след, но пока без живого корпуса речей.

**Scale coding before type.**

- Соционика: `Se`/action-context поддерживает `СЛЭ`, но `СЭЭ` и `ЭИЭ` сильны как альтернативы — личное влияние и эмоционально-смысловая мобилизация могут выглядеть так же.
- Психософия: `1В` вероятна, но миссионерская роль сама требует воли; `2Л` vs `2Э` нужно проверять по материалам групп/семинаров.
- Темпористика: Future/Present выглядит из миссии и действия в месте; возвращение в Мариуполь также может быть Past/Place trace.

**V2-итог.** `СЛЭ / ВЛФЭ / БНПВ`, уверенность `низкая`.

### Василий Фурта

**Method scout.** Только официальная биография; прежняя гипотеза `ВФЛЭ` слишком опиралась на физкультурно-практический след.

**Scale coding before type.**

- Соционика: `ЛСЭ` остаётся за педагогикой, миссионерской организацией, лидерской подготовкой и областным руководством. `СЭЭ` возможно при более личностно-влиятельном живом стиле.
- Психософия: по новым правилам нельзя ставить 2Ф только из профессии учителя физкультуры или практичности. Сильнее методически выглядит `ВЛФЭ`, а `ВФЛЭ` остаётся альтернативой.
- Темпористика: Future/Present связано с миссией и подготовкой лидеров, но только роль.

**V2-итог.** Психософия меняется: primary `ВЛФЭ`, альтернатива `ВФЛЭ`. Остальное без уверенного повышения.

### Николай Романюк

**Method scout.** Есть официальная биография и церковный сайт ИБЦ, где видна экосистема проповедей, групп, ресурсов, консультирования, подготовки к браку. Это даёт больше контекста, но не заменяет анализ конкретных проповедей.

**Scale coding before type.**

- Соционика: `ЭИИ` поддерживается зрелостью, духовной жизнью и пасторской помощью людям; `ЛИИ` остаётся почти равной альтернативой из-за филологии, церковной истории, учительства и системного образования.
- Психософия: `ЛЭВФ` остаётся, но `ЭЛВФ` может стать primary, если в проповедях первична эмоционально-пасторская настройка.
- Темпористика: `ПВ` как история/зрелость + вечностная рамка; `ВП` если богословский смысл первичнее истории созревания.

**V2-итог.** Сохранить `ЭИИ / ЛЭВФ / ПВБН`, но явно считать `ЛИИ` сильной альтернативой. Уверенность `низкая-средняя`.

### Владимир Омельчук

**Method scout.** Только официальная биография. Прежняя `ФВЭЛ` как primary была слишком сильно выведена из ремесленного прошлого и практической церковной жизни.

**Scale coding before type.**

- Соционика: `ЭСЭ` остаётся по общинной атмосфере и практическому пасторскому запуску мест; `СЭЭ` если больше личного напора и внешнего влияния.
- Психософия: по новым правилам нельзя делать 1Ф из ремесла. Запуск новых церквей и переходы между городами сильнее поддерживают `1В`; практический слой может быть `2Ф`. Поэтому primary меняется на `ВФЭЛ`, а `ФВЭЛ` становится альтернативой.
- Темпористика: методически stronger `БНПВ`: future/church-planting direction + present/place. Ранее `НБПВ` может быть эффектом общинной конкретности.

**V2-итог.** Психософия и темпористика меняются: `ВФЭЛ`, `БНПВ`; уверенность низкая.

### Сергей Сизоненко

**Method scout.** Только официальная биография, но образовательные темы довольно ясны: гомилетика, герменевтика, библиология, история церкви, история евангельского движения.

**Scale coding before type.**

- Соционика: `ЛИИ` остаётся из-за структурно-богословского и историко-образовательного следа; `ЛСЭ` если пасторско-организационный стиль окажется практическим.
- Психософия: `ЛВЭФ` остаётся: 1Л + 2В; `ВЛЭФ` если первичнее лидерское направление.
- Темпористика: Past/Future из истории + church planting, но это учебные темы и служебный путь, не личный heatmap.

**V2-итог.** Без смены primary, уверенность `низкая`.

### Владимир Шемчишин

**Method scout.** Официальная биография богата событиями, но почти полностью должностная: долгий путь служения, пасторство, регион, семинария, Союз. По новому gate это не даёт `низкая-средняя`.

**Scale coding before type.**

- Соционика: `ЛСЭ` vs `ЛСИ` остаются: практическая организация vs нормативно-структурная дисциплина. Без живых материалов не различить.
- Психософия: `ВЛ` vs `ЛВ` неразличимы: долгая руководящая нагрузка может быть ролью.
- Темпористика: `ПБ` выглядит как преемственность + развитие институций, но это официальный маршрут.

**V2-итог.** Primary не меняется, но уверенность понижена с `низкая-средняя` до `низкая`.

## Итоговые изменения относительно предыдущей таблицы

1. **Валерий Антонюк:** `ЛСИ` поднят как более сильная альтернатива рядом с `ЭСИ`; уверенность остаётся низкой.
2. **Геннадий Мохненко:** уверенность понижена до низкой из-за конфессиональной оговорки и медийного role-overfit.
3. **Василий Фурта:** психософия изменена с `ВФЛЭ` на `ВЛФЭ`, потому что 2Ф была слишком выведена из практической/физкультурной биографии.
4. **Владимир Омельчук:** психософия изменена с `ФВЭЛ` на `ВФЭЛ`; темпористика — с `НБПВ` на `БНПВ`, потому что запуск новых церквей сильнее поддерживает Волю/Будущее, чем ремесло/место как primary.
5. **Владимир Шемчишин:** уверенность понижена до низкой: богатая биография всё ещё в основном должностная.
6. **Сергей Санников, Николай Романюк, Игорь Бандура:** primary сохранены, но с более явными methodological limits.
