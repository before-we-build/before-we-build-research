# Агентный пайплайн для типирования публичных фигур и критики гипотез

Статус: **исследовательский пайплайн для повышения качества гипотез**, не механизм окончательного типирования. Цель — не заменить живое интервью и не выдать «истинный тип», а уменьшить произвол, отделить источники от интерпретации, собрать альтернативы и показать, где уверенность завышена.

Пайплайн нужен для страниц вроде [[baptist-audience-public-figure-typing-ru]], где публичные баптистские / евангельские фигуры Украины типируются по открытым биографическим следам. Материал должен оставаться в исследовательском слое и не становиться публичным способом «типировать пасторов».

Методический аудит актуальных интернет-подходов см. в [[public-figure-typing-methodology-audit-ru]]. Его главный вывод: пайплайн должен использовать тесты, LLM-анализ и типировочные сообщества только как вспомогательные источники, а основную силу давать multi-source evidence coding, interview/sermon/book traces, шкалам/осям до финального типа, независимым типировщикам и обязательной критике role-overfit.

## Базовый принцип

Один агент почти всегда смешивает три вещи:

1. факты биографии;
2. впечатление от роли / должности;
3. типологическую интерпретацию.

Поэтому пайплайн разделяет роли:

```text
источники → признаки → независимые типировщики → критики → арбитр → журнал уверенности
```

Каждый агент получает ограниченную задачу и должен явно отличать:

- подтверждённый источник;
- наблюдаемый поведенческий / речевой след;
- типологическую интерпретацию;
- альтернативное объяснение;
- роль, которая может имитировать тип.

## Методический слой: что считается перспективным источником

Перед типированием агент должен не только собрать источники, но и оценить методическую силу каждого источника.

| Источник / метод | Использование | Ограничение |
|---|---|---|
| Официальная биография | факты, роли, траектория | слабое доказательство типа; высокий риск role-overfit |
| Полное интервью | стиль выбора, конфликта, аргументации, самоописания | самопрезентация и жанр |
| Полная проповедь / лекция | структура объяснения, повторяющиеся темы, стиль обращения | церковный жанр и тема могут имитировать тип |
| Книги / статьи | устойчивый способ мысли, понятийная архитектура | редактура, жанр, предмет исследования |
| Соционические интервью / Model A / дихотомии | сильнее простого описания типа | зависит от интервьюера и интерпретации |
| Talanov-style шкалы и онлайн-тесты | список признаков и шкальная логика | нельзя применять к публичной фигуре без самопрохождения |
| Afanasyev / Anette / AP-подобные опросники | психософские оси и вопросы | community-инструменты без достаточной независимой валидности |
| Темпористический aspect × position heatmap | обязательный способ не гадать четырёхбуквенный код напрямую | слабая внешняя методическая база; confidence lower by default |
| LLM-анализ текста | извлечение признаков, повторяющихся тем, альтернатив | не финальный типировщик; риск жанрового и авторского bias |

Практическое правило: сначала кодировать **шкалы / оси / признаки**, и только потом собирать тип. Для соционики — информационные аспекты и Model A-сигналы; для психософии — верх/низ, процесс/результат, доминирование/подстройка и признаки 1/2/3/4 позиции; для темпористики — salience каждого аспекта и позиционный режим Target/Creative/Painful/Blind.

## Обязательные ограничения

- Типирование публичной фигуры — гипотеза о повторяющихся паттернах, а не утверждение о сущности человека.
- Использовать формулу проекта: **типовые характеры, не типы человека**.
- Не делать духовных, моральных, пасторских, брачных, церковно-кадровых или доверительных выводов из типологии.
- Не повышать уверенность выше `средняя`, если нет нескольких полных интервью/проповедей/книг и независимой критики.
- Не смешивать уровни:
  - соционика = тактический информационный стиль;
  - психософия = операционный ресурс / распределение внимания и ответственности;
  - темпористика = стратегическая временная / смысловая рамка.

## Входные данные

Для одной фигуры пайплайн должен собрать папку или запись с такими полями:

```yaml
figure_id: romanized-or-slug
name_ru: ""
name_uk: ""
audience_relevance: "почему эта фигура значима для украинской баптистской / евангельской аудитории"
confessional_context: "баптист / евангельский не-баптист / богослов / преподаватель / историческая фигура"
sources:
  - url: ""
    source_type: official_bio | sermon | interview | book | article | encyclopedia | social | video
    access_status: 200 | 403 | 404 | unavailable | manual
    notes: "что именно источник подтверждает"
raw_traces:
  - claim: "наблюдаемый факт или повторяющийся след"
    source_refs: [1, 2]
    strength: strong | medium | weak
    role_overfit_risk: high | medium | low
existing_hypotheses:
  socionics: "если уже есть"
  psychosophy: "если уже есть"
  temporistics: "если уже есть"
```

Минимум для слабого прохода:

- 1 официальная биография;
- 1 длинное интервью, проповедь или текст, где человек не только исполняет должность, но рассуждает о выборе, конфликте, будущем, прошлом, ответственности или смысле.

Минимум для более уверенного прохода:

- официальная биография;
- 3+ полных речевых материала разных лет;
- 1 материал в кризисном / спорном контексте;
- внешняя критика гипотезы или независимое второе типирование.

## Роли агентов

### 0. Агент методического скаутинга

Задача: перед типированием определить, какие современные методики применимы к конкретной фигуре и источникам.

Выход:

```markdown
## Method scout

### Available evidence types
- official biography / interview / sermon / book / article / social / video

### Applicable methods
- Socionics: Model A aspect coding / dichotomies / interview-style differentiators / Talanov-style scale hints
- Psychosophy: position axes / Afanasyev-Anette-AP question families / interview differentiators
- Temporistics: aspect × position heatmap / temporal narrative coding
- LLM/text analysis: trace extraction only, not direct type

### Method limits
- what cannot be inferred from current sources
```

Запрет: не выбирать тип. Этот агент только задаёт методический план и ограничения.

### 1. Агент-источниковед

Задача: найти и структурировать источники, не типировать.

Выход:

```markdown
## Source dossier

### Confirmed facts
- факт — источник

### Repeated themes
- тема — источники

### Role constraints
- что может быть следствием должности, жанра проповеди, церковной культуры, войны, публичной осторожности

### Missing evidence
- каких материалов не хватает для типирования
```

Запреты:

- не писать типы;
- не делать психологических выводов;
- не использовать поисковые сниппеты как факт без перехода к источнику.

### 2. Агент поведенческих признаков

Задача: перевести источники в наблюдаемые признаки без типологических ярлыков.

Если есть длинные тексты, книги, проповеди или интервью, этот агент может использовать LLM/text-analysis подход как вспомогательный слой: выделить повторяющиеся темы, структуры аргументации, временные рамки, типичные глаголы действия, способы обращения к аудитории. Но каждый вывод должен ссылаться на конкретные фрагменты или источники; LLM не должен напрямую решать тип.

Выход:

```markdown
## Observable traces

| Trace | Evidence | Possible non-type explanation | Strength |
|---|---|---|---|
| ... | ... | должность / жанр / культура / возраст / кризис | weak/medium/strong |
```

Примеры признаков:

- как человек строит объяснение;
- повторяет ли историю, будущее, смысл, место/среду;
- ведёт ли через порядок, идею, личное влияние, атмосферу, практическую заботу;
- как говорит о конфликте, ответственности, поколениях, миссии, памяти, будущем.

### 3. Соционический типировщик

Задача: дать 2–3 соционические гипотезы с аргументами и контраргументами.

Выход:

```markdown
## Socionics hypothesis

Primary: XXX
Alternatives: YYY, ZZZ

### Evidence for primary
- trace → interpretation

### Evidence against primary
- trace → why it may not fit

### Differentiators to check
- question/material that would separate XXX from YYY

Confidence: low | low-medium | medium
```

Запрет: не выводить соционику из психософии, должности или темперамента.

Дополнительное требование после методического аудита: сначала дать промежуточный профиль по информационным аспектам / Model A-сигналам / дихотомическим дифференциаторам, а уже потом тип. Онлайн-тесты и Talanov-style inventory можно использовать только как перечень признаков и вопросов, если публичная фигура сама не проходила тест.

### 4. Психософский типировщик

Задача: дать 2–3 гипотезы по порядку Воли, Логики, Эмоции, Физики.

Особенно проверять:

- где человек задаёт стандарт и не нуждается в обсуждении;
- где он процессно помогает другим;
- где заметна уязвимость / болезненная проверка;
- что легко отдаёт на внешнюю рамку.

Выход аналогичен соционическому агенту.

Запрет: не путать служебную ответственность с 1В без проверки; пасторская эмоциональность не равна 1Э; богословское образование не равно 1Л.

Дополнительное требование после методического аудита: явно кодировать три оси — верх/низ, процесс/результат, доминирование/подстройка — и признаки позиций 1/2/3/4 для каждого аспекта. Afanasyev / Anette / Attitudinal-Psyche-подобные материалы можно использовать как источники вопросов и осей, но не как валидированное доказательство.

### 5. Темпористический типировщик

Задача: дать 2–3 гипотезы по Прошлому, Настоящему, Будущему, Вечности.

Проверять:

- Прошлое: идентичность, память, преемственность, история, опыт;
- Настоящее: место, среда, текущая вовлечённость, «здесь»;
- Будущее: миссия, цель, стратегия, поколения, следующий этап;
- Вечность: смысл, богословская рамка, конечная перспектива, призвание.

Запрет: не считать любую христианскую речь о Боге автоматическим 1Вечности; у служителя это может быть жанр и вера, а не темпористическая позиция.

Дополнительное требование после методического аудита: строить `aspect × position` heatmap до выбора типа. Для публичных христианских фигур Вечность почти всегда имеет жанровое загрязнение, поэтому темпористическая уверенность по умолчанию ниже, пока не видно позиционного режима: сам задаёт рамку, процессно развивает, болезненно проверяет или берёт как ресурс.

### 6. Агент-критик роли и переобучения

Задача: атаковать гипотезы через альтернативные объяснения.

Критические вопросы:

- Это тип или должность?
- Это личный стиль или церковный жанр?
- Это устойчивый паттерн или единичный источник?
- Это след войны/кризиса/возраста/поколения?
- Не смешаны ли соционика, психософия и темпористика?
- Не повышена ли уверенность из-за симпатии/известности?
- Не принял ли агент результат интернет-теста, текстового AI-анализа или описания типа за доказательство?
- Есть ли source-quality weighting: official bio, interview, sermon, book, edited text, social media?

Выход:

```markdown
## Critique

### Overfit risks
- ...

### Weak evidence
- ...

### Alternative hypotheses that deserve equal status
- ...

### Confidence downgrade / upgrade
- from X to Y because ...
```

### 7. Агент богословско-пасторской безопасности

Задача: проверить, что итог не превращается в духовный ярлык.

Проверяет, что текст:

- не оценивает духовность;
- не выводит пригодность к служению;
- не делает моральный диагноз;
- не предлагает публично маркировать пасторов;
- не заменяет Писание, церковь, молитву, зрелый совет и плод жизни.

### 8. Арбитр / редактор итоговой гипотезы

Задача: собрать финальную запись, понизить уверенность при споре и сохранить альтернативы.

Правила арбитража:

- если типировщики расходятся сильно — итог `низкая`, не выбирать победителя силой;
- если два независимых агента дают близкую гипотезу, но критик нашёл роль-overfit — максимум `низкая-средняя`;
- если есть 3+ материалов разных лет и критика не разрушила гипотезу — можно `средняя`;
- `высокая` не использовать для публичных фигур без интервью/самоотчёта/внешней типировочной проверки.

## Шкала уверенности

| Уровень | Условие |
|---|---|
| очень низкая | есть только должность, короткая биография или впечатление |
| низкая | есть официальная биография и 1–2 материала, но признаки могут быть ролью |
| низкая-средняя | есть несколько независимых материалов и устойчивая картина, но альтернативы сильны |
| средняя | есть материалы разных лет, разные жанры, критика и явные дифференциаторы |
| высокая | не использовать для текущих публичных биографических типирований без живого интервью и независимой проверки |

Дополнительные gates:

- если есть только официальная биография — максимум `низкая`;
- если есть биография + проповеди одного жанра — максимум `низкая-средняя`;
- если есть тест/самоотчёт без интервью и внешней критики — максимум `низкая-средняя`;
- если тип основан на LLM-анализе текста без ручной проверки цитат — максимум `низкая`;
- если должность сама требует соответствующего поведения, критик обязан рассмотреть downgrade.

## Практическая агентная схема для Hermes

Один запуск на фигуру:

```text
1. Source scout: собрать 5–10 источников, выписать только факты и цитируемые следы.
2. Trace extractor: превратить факты в таблицу наблюдаемых признаков.
3. Socionics agent: типировать только соционику.
4. Psychosophy agent: типировать только психософию.
5. Temporistics agent: типировать только темпористику.
6. Critic agent: атаковать все три гипотезы, искать роль-overfit и слабые места.
7. Safety agent: проверить богословско-пасторские ограничения.
8. Arbiter: собрать итоговую запись и обновить страницу.
```

Если доступно несколько фоновых агентов, независимые роли 3–7 можно запускать параллельно после этапов 1–2. Если доступен только один агент, роли выполняются последовательно, но ответы сохраняются раздельно.

## Промпты ролей

### Source scout prompt

```text
Ты агент-источниковед для Before We Build research. Твоя задача — собрать источники о публичной баптистской/евангельской фигуре и выписать только проверяемые факты и наблюдаемые следы. Не типируй. Не делай психологических выводов. Отделяй официальный источник, интервью, проповедь, книгу, энциклопедию и медиа. Отмечай, что может быть следствием должности или жанра. Верни source dossier с URL, статусом доступа, фактами, повторяющимися темами и недостающими материалами.
```

### Typologist prompt

```text
Ты типировщик только по одной системе: [Socionics / Psychosophy / Temporistics]. Используй только предоставленный source dossier и observable traces. Не выводи тип из должности напрямую. Дай primary hypothesis, 1–2 alternatives, evidence for, evidence against, differentiators to check, confidence. Если данных мало, честно понизь уверенность.
```

### Critic prompt

```text
Ты критик типировочных гипотез. Твоя задача — не предложить красивый тип, а сломать слабую аргументацию. Проверь role-overfit, смешение систем, жанровые эффекты, недостаток источников, альтернативные объяснения, завышенную уверенность. Верни список проблем, альтернативы, которые должны остаться, и рекомендацию по снижению/повышению уверенности.
```

### Safety prompt

```text
Ты богословско-пасторский safety reviewer для Before We Build. ```text

Понизь уверенность при споре. Не делай финального вердикта. Используй формат: открытые следы, гипотезы по трём системам, критика и риск переобучения, что проверить дальше.

## Как применить к текущей странице

1. Взять одну фигуру из [[baptist-audience-public-figure-typing-ru]].
2. Создать рабочий dossier в `wiki/sources/` или временный исследовательский черновик.
3. Прогнать роли пайплайна.
4. Обновить запись фигуры: добавить `Критика и риск переобучения`, уточнить альтернативы и уверенность.
5. Если по нескольким фигурам изменились типы, обновить краткую таблицу.
6. Записать изменения в `log.md`.

## Приоритет для следующего применения

Лучше всего начать с фигур, где есть больше материалов и влияние на аудиторию высокое:

1. Сергей Санников — много богословских/исторических материалов, можно точнее разделить ЛИИ/ИЛИ и ВПБН/ВБПН.
2. Николай Романюк — пастор-учитель, возможно есть много проповедей; важно отделить ЭИИ от ЛИИ.
3. Валерий Антонюк — официальная роль сильна, поэтому нужен критик role-overfit.
4. Игорь Бандура — нужна проверка ЭИЭ/ЭСЭ и 1Э/1В.
5. Ярослав Пиж / Валентин Синий / Тарас Дятлик — добавить после полноценного source dossier.
