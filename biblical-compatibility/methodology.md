# Методологія

## Завдання

Методологія цього модуля — співвіднести Писання, герменевтику, богословську обережність, наявний typology research repository і практичну мудрість для одиноких християн.

## Порядок роботи

1. **Прочитати repository.** Знайти, які ідеї про сумісність, шлюб, стосунки, християнство, церкву й типологічні рівні вже існують.
2. **Не дублювати типологію.** Типологічні описи залишаються у `wiki/` і `raw/`.
3. **Почати з Писання.** Кожна нормативна теза має бути звірена з біблійними текстами.
4. **Позначити рівень авторитету.** Кожне твердження отримує категорію: пряме біблійне вчення / біблійний принцип / пастирське застосування / типологічна гіпотеза / практична порада.
5. **Додати пастирські межі.** Особливо там, де сумісність може стати ідолом, страхом, ярликом або маніпуляцією.
6. **Писати для одиноких християн.** Без тиску до шлюбу і без романтизації самотності.

## Категорії тверджень

| Категорія | Що це означає | Дозволений тон |
|---|---|---|
| Пряме біблійне вчення | Текст прямо навчає це в контексті | «Писання вчить…» |
| Біблійний принцип | Обережний висновок із кількох текстів | «Біблійна мудрість веде нас до…» |
| Пастирське застосування | Практичне застосування до сучасної ситуації | «Мудро / небезпечно / варто…» |
| Типологічна гіпотеза | Нижчий допоміжний рівень з repository | «Модель може допомогти поставити питання…» |
| Практична порада | Досвідна рекомендація без нормативної сили | «Може бути корисно…» |

## Герменевтичні правила

- Не будувати доктрину сумісності на одному вирваному вірші.
- Не переносити старозавітний чи апостольський контекст без пояснення.
- Не робити з мудрісних принципів абсолютні гарантії.
- Не використовувати Писання для тиску на одиноких.
- Не підміняти любов комфортом, а святість — психологічною легкістю.
- Не робити типологію прихованою владою під біблійною мовою.


## Джерела integration з typology research

Цей модуль не переказує типології з нуля. Для технічної бази він посилається на:

- `wiki/concepts/compatibility-level-boundaries-uk.md` — стратегічний / операційний / тактичний рівні.
- `wiki/concepts/typologies-as-supporting-tools-uk.md` — типології як допоміжні інструменти.
- `wiki/concepts/research-layer-vs-practical-guidance-uk.md` — межа між дослідницьким шаром і практичним керівництвом.
- `wiki/concepts/biblical-grounding-policy-uk.md` — правило біблійної опори для нормативних тверджень.
- `wiki/concepts/christian-foundation-of-family-uk.md` і `wiki/concepts/family-formation-principles-uk.md` — первинність віри, характеру, вірності, відповідальності й спільноти.
- `wiki/sources/socionics-detailed.md`, `wiki/sources/psychosophy-detailed.md`, `wiki/sources/temporistics-detailed.md` — технічні описи трьох систем.
- `wiki/concepts/adequate-intertype-formats-ru.md` — ідея формату, дистанції, навантаження і відповідальності як дослідницька гіпотеза, не як вирок.
