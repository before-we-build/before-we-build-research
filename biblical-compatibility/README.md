# Biblical Compatibility

**Biblical Compatibility** — це окремий біблійно-герменевтичний шар у наявному `before-we-build-research` repository. Його мета — не створити нову типологічну базу, а перевірити, обмежити, очистити й богословськи рамкувати ідею сумісності для одиноких християн.

Робоча назва книги: **«Сумісність для одиноких християн»**.

## Головна теза

Сумісність може бути корисною практичною категорією, якщо вона допомагає мудро будувати зв’язки, що ведуть до миру, збудування, служіння і духовного росту. Але сумісність не є вищою за Христа, любов, святість, характер, вірність, Писання і церковну мудрість.

## Що цей модуль робить

1. Аналізує наявний typology research repository без дублювання його змісту.
2. Створює ієрархію авторитетів: Писання → герменевтика → церковна мудрість → спостереження → типологія → особистий досвід.
3. Класифікує твердження про сумісність у `claims-register.md`.
4. Дає герменевтичні нотатки до ключових біблійних уривків у `hermeneutics/`.
5. Створює тематичні богословсько-пастирські файли в `themes/`.
6. Дає перевірений матеріал, з якого окремий book repository `compatibility-for-single-christians` формує чисту рукописну версію.

## Що цей модуль не робить

- Не доводить типологію з Біблії.
- Не називає типологічну модель біблійною антропологією.
- Не дає гарантії шлюбу або майбутнього.
- Не дозволяє знецінити одиноких християн.
- Не створює тиск «обов’язково знайти пару».
- Не робить типи, дари або психологічний комфорт мірилом духовної зрілості.


## Джерела integration з typology research

Цей модуль не переказує типології з нуля. Для технічної бази він посилається на:

- `wiki/concepts/compatibility-level-boundaries-uk.md` — стратегічний / операційний / тактичний рівні.
- `wiki/concepts/typologies-as-supporting-tools-uk.md` — типології як допоміжні інструменти.
- `wiki/concepts/research-layer-vs-practical-guidance-uk.md` — межа між дослідницьким шаром і практичним керівництвом.
- `wiki/concepts/biblical-grounding-policy-uk.md` — правило біблійної опори для нормативних тверджень.
- `wiki/concepts/christian-foundation-of-family-uk.md` і `wiki/concepts/family-formation-principles-uk.md` — первинність віри, характеру, вірності, відповідальності й спільноти.
- `wiki/sources/socionics-detailed.md`, `wiki/sources/psychosophy-detailed.md`, `wiki/sources/temporistics-detailed.md` — технічні описи трьох систем.
- `wiki/concepts/adequate-intertype-formats-ru.md` — ідея формату, дистанції, навантаження і відповідальності як дослідницька гіпотеза, не як вирок.
