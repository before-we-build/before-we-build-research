# Psychosophy Relation Naming Sources

This source note records web-source reconnaissance for Psychosophy / Psyche-Yoga relation naming. It distinguishes classical Russian Psychosophy labels, later Russian technical naming attempts, Before We Build working labels, and Attitudinal Psyche labels.

## Source Status

- **Classical / source-backed tradition**: labels commonly attributed to Afanasyev's *Syntax of Love* tradition and repeated in Russian-language summaries.
- **Russian technical extension**: a later systematic attempt to describe all 24 directional signatures by combinations of four basic interfunctional contacts. This is not treated here as canonical Afanasyev terminology.
- **Before We Build working label**: internal mechanism-first display labels used to reduce romantic, moralized, or deterministic interpretation.
- **Attitudinal Psyche branch**: modern English-language derivative system with its own relation names. Useful for comparison, but not automatically equivalent to classical Psychosophy.

## Sociomodel Summary

Source: `https://sociomodel.ru/psy-intertype.htm`

The page summarizes four main Psychosophy intertype relation types:

- **Филия / Philia** — same aspects in all four positions.
- **Квазиэрос / Quasi-eros** — called **псевдофилия / pseudophilia** by Afanasyev in some summaries; 1st and 2nd functions cross, and 3rd and 4th functions cross.
- **Эрос / Eros** — crossing of 1st and 3rd functions.
- **Агапэ / Agape** — crossing of 2nd and 3rd functions, and also 1st and 4th functions in the full form.

The page also states that other cases can be reduced to combinations of separate manifestations of the basic intertype relations for individual function pairs.

## Typologies.ru / 4X_Pro Technical Extension

Source: `https://typologies.ru/psycheyoga/intertype.htm`

The article “Интертипные отношения с точки зрения межфункционных взаимодействий” proposes two notations for Psychosophy intertype relations:

1. A four-digit signature notation, where each digit indicates the position of the same aspect in the partner's type, read from the first partner's 1st to 4th function.
2. A four-letter relation-composition notation using:
   - `F` — филия / Philia / identity of components;
   - `Q` — квазиэрос / Quasi-eros / component dampening;
   - `E` — эрос / Eros / target-component complement;
   - `A` — агапэ / Agape / resource-component complement.

The article gives 24 possible signatures. Some have informal names; many remain technical combinations.

## Attitudinal Psyche Relation Names

Source: `https://www.attitudinalpsyche.com/theory/type-relations/`

Attitudinal Psyche is a modern English-language system derived from / related to Psychosophy. It has its own named relation system. These labels should be treated as **Attitudinal Psyche terminology**, not as automatically canonical Russian Psychosophy terminology.

Relation headings found on the source page:

- Shared Sexta
- Dual
- Identical
- Solution
- Sister
- Square
- Radiance
- Instruction
- Triangular
- Invention
- Assistance
- Enhancement
- Regulation
- Linear
- Near-Identical
- Cousin
- Customary
- Specificity
- Opposed Sexta
- Faux-Identical
- Suffocation
- Conflict

Before We Build may use these names only with explicit source labeling, e.g. “Attitudinal Psyche calls this relation X,” and should not collapse them into Afanasyev's Greek relation labels or into Socionics relation names.

## Before We Build Editorial Rule

When displaying Psychosophy relation labels:

1. Keep **Agape, Philia, Eros, Pseudophilia / Quasi-eros** as tradition-aware labels.
2. For 24-signature technical labels from Typologies.ru, preserve them as **Russian technical extension labels** and pair them with mechanism descriptors.
3. For Attitudinal Psyche labels, mark them as **AP branch terminology**.
4. For product-facing output, prefer mechanism-first names with caveats over romantic or destiny-sounding names.
5. Never infer total relationship outcome from a Psychosophy relation label alone.
